CSS - Media Queries

A media query (or media query) consists of a media type, and at least one expression that limits the scope of CSS declarations, taking advantage of media features such as width, height, or color display. Added in CSS3, the media queries allow to adapt the presentation of the content to a wide range of devices without changing the content itself.

We can change the media queries according to:

  • Type of media (screen, print, ...)
  • Device Width/Height
  • Device Orientation

Example - Change the width of the window

I will have a different background at each break point

					/*DEFAULT STYLES (desktop or mobile)*/
					#box {
						width: 100%;
						padding: 3em 10%;
						text-align: center;
						background: #ccc;
					}

					/* Break Point 1 - Landscape Tablets  */
					@media only screen and (max-width: 1024px) {

						#box {
							background: #AED6F1;
						}

					}


					/* Break Point 2 - portrait tablets  */
					@media only screen and (max-width: 768px) {

						#box {
							background: #A569BD;
						}

					}


					/* Break point 3 - landscape phones  */
					@media only screen and (max-width: 667px) {

						#box {
							background: #CB4335;
						}

					}


					/* Break point 4 - phones portrait  */
					@media only screen and (max-width: 480px) {

						#box {
							background: #2E4053;
							color: #fff;
						}
						
					}
