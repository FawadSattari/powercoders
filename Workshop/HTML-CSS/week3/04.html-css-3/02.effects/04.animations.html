CSS - animation

CSS animations make it possible to animate transitions from one CSS style configuration to another.

Animations consist of two components, a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animationâ€™s style, as well as possible intermediate waypoints.

animation: [animation-name] [animation-duration] [animation-delay] [animation-iteration-count] [animation-delay] [animation-fill-mode];

Animation 1


        #c1 {
          opacity: 0;
          animation-duration: 5s;
          animation-name: anim-c1;
          animation-delay: 2s; /*starts after 2 seconds*/
          animation-fill-mode: forwards; /*stop at the end of anim*/
          animation-direction: alternate; /*animates back and forth*/
        }
        @keyframes anim-c1 {
          from {
            width: 100px;
            opacity: 0;
          }
    
          to {
            opacity: 1;
            width: 50%;
          }
        }

Animation 2


            #c2 {
              /*Box defaults*/
              opacity: 0;
              position: absolute;
              right: 20px;
        
              /*Animation shortcut properties*/
              animation: slide-bottom 3s cubic-bezier(0.250, 0.460, 0.450, 0.940) both;
            }
            
            @keyframes slide-bottom {
              0% {
                opacity: 0;
                top: -200px;
              }
              100% {
                opacity: 1;
                top: 50%;
                transform: translateY(-50%);
              }
            }
